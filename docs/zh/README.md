### 动机

我需要一个轻量级的、易于使用的、稳定的分布式配置中心，但我不希望它过多地影响我的应用程序的性能。
虽然有很多开源配置中心，但很少有人用Golang编写，所以我想试一试。

### 特色

* 使用golang编写
* 轻量
* 支持命名空间
* 支持一定程度的灰度
* 存储层插件化
* 稳定
